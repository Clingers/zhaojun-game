# 第七章：暴风雪

> **状态**：✅ 定稿（老板审阅通过，2026年7月）
> 
> **主题**：暴风雪。寻找避风处，画面逐渐变白
> 
> **色调**：惨白 + 冷灰（随进度逐渐变白）
> 
> **音乐**：暴风雪呼啸 + 呼吸急促声（昭君） + 极简低音
> 
> **时长**：约 2-3 分钟
> 
> **收藏品**：雪花

---

## 完整剧本

```
【开场】 淡入
草原的白天，但天是灰白色的。
风比昨天更大。
远处的天边有一道黑线——暴风雪来了。
【音效】 风声（比第六章更大，带呼啸感）

【音乐起】 极简低音，持续，压抑

【镜头1】 马车在风里
风把草吹得伏在地上，像海。
马车走得极慢。
昭君在车里，裹着厚衣服。
她的呼吸在冷空气里变成白雾。
【音效】 呼吸声（轻微，但清晰）

【叙事文本·淡入】
风是从凌晨开始变大的。
马夫说前面可能有雪。
昭君没有说什么。
她知道这是必经之路。
只是没有想到会来得这么快。

【互动点1：看窗外】
（玩家点击车窗）
外面已经是白茫茫一片。
风卷着雪沫，打在玻璃上。
看不清路。看不清天。
只有白，和风的声音。
【音效】 风雪呼啸声加强

昭君（独白）：
"这就是漠北的冬天。"

【叙事文本】
马夫的声音（帘子外，几乎被风吞掉）：
"姑娘，我们得找地方避一避。"
"前面有一个破旧的帐子，不知道还能不能住。"

昭君：
"去那里。"

【互动点2：寻找避风处】
（玩家点击屏幕上的"寻找避风处"按钮，或点击帐篷方向）
马车向远处的黑点驶去。
风越来越大。
呼吸声越来越重。
【音效】 风雪声持续加强，呼吸声逐渐明显

【互动点3：点击帐篷】
（玩家点击帐篷）
帐篷还在，但破了一个角。
风从破洞钻进来，发出呜呜的声音。
昭君下车，快步走进去。
里面很暗，有干草的味道。
角落有一张旧毡毯，磨得发白，还有破洞。
她蹲下来，先把毡毯拖到中间，然后找了几块布把破洞的地方塞住。
风的声音小了一点。
【音效】 风声忽然减弱（对比强烈）

昭君（独白）：
"找到了。"

【互动点4：点击雪地】
（玩家点击帐篷外的雪地）
昭君蹲下来，用手接了一片雪花。
雪花在她掌心融化，变成一滴水。
她看着那滴水，看了很久。
【音效】 细微的雪落声

【收集触发】
玩家点击雪花 → 收集"雪花"
（后台：collectibles.snowflake.unlocked = true）

【叙事文本】
她站起来，拍了拍身上的雪。
帐篷里很冷，但她能喘气了。
马夫在外面收拾马车。
风还在吹，但帐篷挡住了大部分。
她坐下来，背靠着木杆。
闭上眼睛。

【音效变化】
风雪声在帐篷外持续，但里面相对安静。
呼吸声慢慢平复。
极简低音继续，但音量降低。

【结尾·闭眼睁眼】
她睁开眼睛。
帐篷里还是暗的。
但外面不一样了——天黑了。
有火光在帐篷外面跳动。
篝火的声音噼啪响。
匈奴人的笑声和说话声，从很远的地方飘过来。
【音效】 篝火声 + 人声（远处，模糊）

昭君坐起来。
掀开帐篷的门帘，向外看。
外面已经点起了篝火。几个匈奴人围坐在火堆旁边。
看见她，有人挥了挥手。
她也挥了挥手。
火光照着她的脸，暖的。

【章节结束】
镜头拉远——帐篷、篝火、草原的夜空。
【字幕】
第七章·暴风雪  完

【界面】
▶ 继续前行（进入第八章·篝火）
🎵 查看旅行册
```

## 设计亮点

1. **画面逐渐变白**：本章的视觉核心——从草原的蓝绿→灰白→惨白，暴风雪来临时画面几乎全白，只在屏幕边缘留一点灰
2. **声音的动态变化**：风声持续加强 → 找到帐篷后骤然减弱 → 雪停后的寂静。用声音表现"找到避风处"的释放感
3. **呼吸声**：她的呼吸从急促到平复，是本章的另一个"进度条"——玩家能直观感受到她的身体状态
4. **帐篷的象征意义**：这一章的核心意象——"找到一个能暂时躲避的地方"。对于昭君来说，匈奴就是她的"帐篷"，起初是破的，但她慢慢适应
5. **雪花收集**：最轻的收集品，只有掌心那么大，转瞬即逝——对应她在极端环境中的脆弱和坚韧
6. **连续叙事衔接**：本章结束于"闭眼睁眼"——她睁开眼睛发现外面天黑了，篝火已经生起。**时间跳转到夜晚，无缝衔接第八章·篝火**
