# 第五章：雁门关

> **状态**：✅ 定稿（老板审阅通过，2026年7月）
> 
> **主题**：离开汉朝。决绝、界限
> 
> **色调**：铁灰 + 沙黄
> 
> **音乐**：大风声（城门洞呼啸）。**其他音乐完全停止**。
> 
> **时长**：约 2-3 分钟
> 
> **收藏品**：雁羽

---

## 完整剧本

```
【开场】 淡入
灰色的天空。没有云，就是灰。
风很大，吹起地上的沙土。
远处出现一座关城——雁门关。
城楼很高，城门很厚。
【音效】 风声（比之前任何章节都大，呼啸感）

【音乐】 完全停止。只有风声。

【镜头1】 马车向关门驶去
路是土路，很硬，车辙很深。
两侧是山壁，颜色发暗。
风从山壁间吹过来，发出呜呜的声音。
【音效】 风声持续，无音乐

【叙事文本·淡入】
这是汉朝最后一个关隘。
过了这道门，就离开中原了。
昭君没有看关城。
她一直看着窗外，但目光没有焦点。

【互动点1：最后看一眼汉地】
（玩家点击车窗触发）
昭君回望——南方。
什么都有没有。山、路、风、沙。
天是灰的，地和天连在一起。
【音效】 风声忽然变大

昭君（独白）：
"这就是……最后一程了。"

【互动点2：触碰关门】
（玩家点击关门，或马车停稳后自动触发）
昭君下车。
她走到关门前。
门是铁灰色的，厚木外包铁皮。
上面有划痕，有风化的痕迹。
她伸手摸了摸门板。
冰凉，粗糙，有铁锈的味道。
【音效】 风声在城门洞里形成回音，呼啸

昭君（独白）：
"这道门关上了，就再也打不开了。"

【叙事文本】
关门开始移动。
很慢。铰链发出低沉的声音。
风从门缝里钻进来，呜呜响。

【闪回·极短（2秒）】
（画面一闪）
长安·未央宫。
元帝站在殿前，远眺宫门方向。
风把他的袍子吹得轻轻摆动。
他站着，没有动。
【闪回结束】
【镜头回到雁门关】
关门还在移动。

【互动点3：最后一片雁羽】
（玩家点击空中飘落的雁羽）
一只大雁从关城上方飞过。
翅膀的影子在城墙上滑过。
雁羽飘落，在风里打转。
昭君伸手，接住了它。
【音效】 风吹雁羽的沙沙声

昭君（独白）：
"大雁。"
"它们还会再来。"
"而我……"

（停顿。没有说完。）

【收集触发】
玩家点击雁羽 → 收集"雁羽"
（后台：collectibles.wild_goose_feather.unlocked = true）

【叙事文本】
关门关上了。
很慢，很沉。
风从门缝里最后钻进来，然后断了。
关门彻底合拢。
没有轰隆声。只有沉重的、缓慢的视觉过程，配合风声的骤然减弱。
然后归于寂静。
只有风。

昭君站在关门前，看了很久。
她没有哭。
就是站着，看那道门。

【结尾·进入关内】
昭君转身，向关内走去。
马蹄声响起。
车队启动，跟着她进了城。
【镜头·回望】
从关内看向关门——那道门已经关上，只剩下一片铁灰色的墙。
天还是灰的，风还是吹。

【章节结束】
关门在视野里彻底消失。
【字幕】
第五章·雁门关  完

【界面】
▶ 继续前行（进入第六章·草原）
🎵 查看旅行册
```

## 设计亮点

1. **完全静音（除风声外）**：本章是所有章节中音乐最少的——只有风声。象征"离开汉朝"，一切繁华丝竹都停了。
2. **"关门"的两次触**：玩家可以触摸关门（冰凉）→ 看它关上（缓慢、沉重、无声）。关门从物理→视觉，两层。
3. **雁羽与"落雁"传说的呼应**：大雁飞过关城，昭君接住雁羽——她知道自己再也回不来，大雁可以。
4. **元帝闪回（2秒）**：极短，元帝在殿前远眺——两人相隔千里，只有风把他们的思绪连在一起。
5. **"而我……"未说完**：台词在这里中断，不说完"我不会了"。留白。
6. **关门的音效处理**：**无轰隆声**。关门是缓慢的、沉重的视觉过程，配合风声骤然减弱。安静比声音更有力量。
