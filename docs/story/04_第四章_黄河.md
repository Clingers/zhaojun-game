# 第四章：黄河

> **状态**：✅ 定稿（老板审阅通过，2026年7月）
> 
> **主题**：壮阔、考验
> 
> **色调**：赭石 + 浪白
> 
> **音乐**：黄河水奔腾声 + 鼓（小游戏时加强）
> 
> **时长**：约 2-3 分钟
> 
> **收藏品**：黄河砂

## 完整剧本

```
【开场】 淡入
黄土高原。天很高，云很厚。
黄河在远处，像一条黄色的带子，又宽又浊。
【音效】 风声（开阔） + 远处隐约的水声（沉闷）

【镜头1】 马车停在岸边
渡口。有几艘木船，不大。
马夫在和船夫说话。
船夫摇头，说了什么。马夫皱眉。

【音乐起】 鼓点，缓慢，沉重（但未进入游戏状态）

【叙事文本·淡入】
黄河的水比渭河要猛得多。
船夫说今天浪大，不安全。
马夫在想办法。

昭君从车上下来，走到岸边。
她从来没有见过这样的河。
不是清的那种，是浑的，黄褐色的，一眼望不到对岸。
水在流，一直在流，好像永远流不完。

【互动点1：点击河水】
（玩家点击河面）
昭君蹲下来，伸手摸了摸水面。
水是凉的，很冲，像有什么东西在底下拽。
她缩回手。

昭君（独白）：
"黄河的水，比长安的井水要浑。"

【叙事文本】
马夫走过来。
"姑娘，船走不了。"
"我找人来帮忙。"

【小游戏触发】
（玩家点击"帮忙过河"按钮，或点击马夫）
马夫带着几个人过来了，都是当地的壮汉。
他们抬着马车，一步一步往水里走。
【音效变化】 鼓声加强，节奏加快
【音乐变化】 鼓成为主导，模拟过河的紧张感

【过河小游戏】
（节奏敲击QTE）
鼓点响起，屏幕上出现一个圆环，沿着固定轨道移动。
鼓点落下的瞬间，圆环与中心重合——玩家点击屏幕。
配合鼓点节奏敲击，让马车稳步过河。

**游戏流程**：
- 壮汉们喊号子（"嗨哟——"）
- 鼓点节奏：咚——咚咚——咚
- 马车晃了一下（玩家需在鼓点处点击"稳住"）
- 水淹过车轮（鼓点变快，持续点击）
- 前面的人滑了一下（连续两次鼓点，玩家快速点击两次）
- 终于上了对岸

**游戏结束**：
马车安全上岸。
鼓声渐弱，回到背景音。
【音效】 喘息声 + 水声

【互动点2：捡起黄河砂】
（玩家点击河滩上的石头）
昭君弯腰捡起一块石头——圆润的，被水冲了很久。
沙子还沾在石头的缝隙里，黄褐色。
她用手擦了擦，石头更亮了。
【音效】 细微的碰撞声

【收集触发】
玩家点击石头 → 收集"黄河砂"
（后台：collectibles.yellow_river_sand.unlocked = true）

【叙事文本】
她拍了拍手上的泥，上车。
衣摆湿了一角，头发也被溅湿了几缕。
她没有管。

【结尾】
马车继续走。
黄河留在身后，像一条黄色的河流，越来越小。
【音乐淡出】
车行至傍晚，天空开始暗下来。

【章节结束】
【字幕】
第四章·黄河  完

【界面】
▶ 继续前行（进入第五章·雁门关）
🎵 查看旅行册
```

## 小游戏设计

**类型**：节奏敲击QTE

**玩法**：
- 屏幕上出现一个圆环，沿固定轨道移动
- 鼓点响起，圆环与中心重合的瞬间，玩家点击屏幕
- 配合鼓点节奏敲击
- 不设失败条件——无论如何都会过河
- 表现"更稳"或"更险"的区别，但不中断剧情

**音效设计**：
- 鼓点节奏：咚——咚咚——咚（缓慢）
- 游戏进行中：节奏逐渐加快
- 过河成功：鼓声渐弱，喘息声+水声

---

## 设计亮点

1. **触感对比**：渭河的水（圆润石头）→ 黄河的水（冲撞感），同样的"水"在不同章节呈现不同质感
2. **小游戏的叙事功能**：不是关卡，而是体验——让玩家感受到过河的紧张和壮汉们的力量
3. **衣摆微湿的细节**：一个微小但真实的身体感受，暗示她已经开始被旅途改变
4. **"黄河砂"的命名**：诗意优先，以"砂"代"石"，呼应黄河的泥沙俱下
