# 第九章：王庭

> **状态**：✅ 定稿（老板审阅通过，2026年7月）
> 
> **主题**：抵达匈奴王庭，见到呼韩邪单于
> 
> **色调**：金色 + 红色（匈奴宫廷的华丽）
> 
> **音乐**：匈奴宫廷音乐（胡笳、鼓，不同于汉宫廷丝竹）
> 
> **时长**：约 2-3 分钟
> 
> **收藏品**：无（本章为"旅行册合成"章节）

---

## 完整剧本

```
【开场】 淡入
不是切换——是**慢慢亮起来**。
上一秒是篝火的暖橙，这一秒慢慢变成金色。
帐篷。但不是第七章那个破帐篷——是更大的、更华丽的帐篷。
毡毯是红色的，绣着花纹。
光线从帐篷顶部的开口透进来，照在灰尘上，像金色的雾。

【音效】 风声（但很弱，帐篷隔音） + 远处的匈奴人说话声（模糊）

【音乐起】 匈奴宫廷音乐（胡笳 + 鼓点，缓慢，庄严）

【镜头1】 帐篷内部
昭君醒来。
她躺在一张毡毯上，身上盖着一张羊皮。
旁边有一个小木几，上面放着一碗奶和一块干粮。
她坐起来，有点懵。
这里不是第七章那个破帐篷。
【音效】 帐篷布料的摩擦声

【叙事文本·淡入】
她不知道睡了多久。
醒来的时候，已经在王庭了。
匈奴人把她移到了这里——更大的帐篷，更暖的毡毯。
她摸了摸身上的羊皮，很软，很厚。
这是她这辈子睡过的最暖和的床。

【互动点1：看四周】
（玩家点击帐篷内部）
帐篷里有红色的柱子，上面刻着图案。
有皮制的家具，形状和汉地的不一样。
光线从顶部透进来，形成一道金色的光柱，里面有灰尘在飘。
【音效】 远处的人声（隐约，忽远忽近）

昭君（独白）：
"这里是……王庭？"
"我已经到了。"

【叙事文本】
帐篷门帘掀开。
一个匈奴女子走了进来——年龄和昭君相仿，穿着皮衣，戴着银饰。
她看见昭君醒了，笑了一下，说了什么。
昭君听不懂。
但她指了指碗，示意她喝。
昭君接过碗，点了点头。

【旅行册合成·动画】
（玩家点击"打开行囊"按钮，或自动触发）
昭君打开随身的行囊。
里面不是普通的衣物——是一本用布帛包着的"册子"。
她解开布帛，一页一页翻开。
【视觉效果】 每一页都是她之前收集的"记忆碎片"：
   - 第一页：铜镜碎片 → 映出长安的宫殿
   - 第二页：圆润的石头 → 渭河的水声
   - 第三页：山树叶 → 秦岭的雨
   - 第四页：黄河砂 → 黄河的水流
   - 第五页：白鹰羽 → 草原的天空
   - 第六页：雪花 → 暴风雪的寒冷
   - 第七页：篝火炭 → 火堆的温度
每一页都有声音——当她翻到那一页时，对应的音效响起。
翻完最后一页，册子合上。
封面上出现四个字，是她的笔迹：
**一路向北**

【音效】 翻页声 + 每一页对应的环境音（流水、雨声、风声等）

昭君（独白，看着册子）：
"原来……已经走了这么远了。"

【叙事文本】
帐篷门帘再次掀开。
一个高大的匈奴男子走了进来——呼韩邪单于。
他穿着华丽的皮裘，戴着鹰羽冠。
看见昭君，他停下脚步，看了她一会儿。
然后他说了一句话，声音低沉，有力。
匈奴语（字幕浮现，随即消失）：
"你终于来了。"
昭君听不懂，但她点了点头。
他也点了点头。
【音效】 匈奴语（低沉，简短，无法翻译）

【会面·极短】
（没有对话，只有目光）
他走近一步，伸出手——不是要碰她，是要看她的脸。
她站着，没有动。
光线从帐篷顶部照下来，照在他们两个之间。
金色的光，红色的毡毯。
几秒钟。
然后他收回手，转身走了出去。
门帘放下。
【音效】 脚步声（远去的）

昭君（独白）：
"他……看起来很累。"

【结尾·时间跳转】
镜头从帐篷门帘望出去——外面是王庭的帐篷群，红色的、蓝色的，在阳光下闪亮。
远处有人在走，马在跑。
这是另一个世界。
【字幕淡入】
第九章·王庭  完

【界面】
▶ 继续前行（进入第十章·北地）
🎵 查看旅行册（已解锁完整版）
```

## 设计亮点

1. **旅行册合成**：本章核心体验——玩家打开行囊，一页一页翻开之前收集的7个碎片，每一页都唤起对应的声音和画面。最后册子封面上出现"一路向北"四个字。
2. **时间跳转的质感**：从"破帐篷"到"王庭帐篷"，不是切换，是慢慢亮起来——金色的光柱、红色的毡毯、灰尘在光里飘。安静而华丽。
3. **呼韩邪单于的会面**：只有几秒钟，只有一句"你终于来了"，然后目光对视。**"他看起来很累"**——她只说了这一句，没有更多评价。
4. **色调的转换**：从第七章的惨白 → 第八章的暖橙 → 第九章的金红。色彩在慢慢变化，像旅程在慢慢深入。
5. **"她到了"的安静**：没有激动，没有眼泪，只是"原来已经走了这么远了"。旅行册合上的那一刻，玩家感受到的是一种完成感。

## 关键设定确认

- [x] 呼韩邪单于外貌：模糊处理（华丽的皮裘，鹰羽冠），保持距离感
- [x] 会面对话：单于说"你终于来了"（匈奴语+字幕），昭君点头
- [x] 旅行册触发顺序：在会面前，打开行囊→翻开7个碎片→合上
- [x] 无新收集品：旅行册合成是本章核心体验
