# 第三章：秦岭

> **状态**：✅ 定稿（老板审阅通过，2026年7月）
> 
> **主题**：下雨，山林，孤寂
> 
> **色调**：墨绿 + 雨灰
> 
> **音乐**：山林雨声 + 远处雷声 + 琵琶慢拨
> 
> **时长**：约 2-3 分钟
> 
> **收藏品**：一片山树叶

---

## 完整剧本

```
【开场】 淡入
浓绿的山林。秋天的树，颜色很深。
天空是铅灰色的。
已经开始下雨了——不大，但密。
雨丝斜斜地打下来，落在树叶上，沙沙响。
【音效】 雨声（打在树叶、树皮上） + 远处隐约雷声

【镜头1】 马车在山路上
路很窄，泥泞。
马车走得慢，马蹄打滑的声音。
【音乐起】 琵琶慢拨，缓慢，孤寂感

【叙事文本·淡入】
雨是从中午开始下的。
昭君问马夫还要走多久。
马夫说翻过前面这座山，就是秦岭南坡。
她没有再问。
雨把一切都浸得很沉。
连远处的鸟叫都听不见了。
只有雨声。

【互动点1：触摸车窗上的雨痕】
（玩家点击车窗玻璃）
雨水顺着玻璃流下来。
昭君的手指在窗内跟着水痕移动，一上一下。
外面是山，绿色的，但已经暗了下来。
雨滴打在玻璃上的声音，很近。
【音效】 雨打玻璃（轻微，但清晰）

【闪回·极短】
（画面忽然闪了一下——不是黑，是白。雨天的白。）
深宫的窗。同样的雨天。
昭君坐在窗边，手托着腮，发呆。
外面雨丝斜斜。
织布机的声音很远。
没有人在她身边。
只有雨。
【闪回结束】
【镜头回到现实】
她还在看雨痕。
手指还停在玻璃上。

昭君（独白，极轻）：
"火小的时候，人也会觉得冷。"
"添一根柴，它就旺了。"
"人也一样吧。"

【中场：停车】
马车停了。
马夫的声音（帘子外，带着雨意）：
"姑娘，前面路太滑，走不动了。"
"我们在这儿歇一晚，等明天再说。"

昭君：
"好。"

【镜头2】 车外
马夫在路边搭了一个简易的篷子。
火堆已经点起来了。
雨还在下，但火堆周围有一小块干燥的地方。
【音效】 柴火噼啪声 + 雨声持续（但减弱）

【互动点2：添柴】
（玩家点击火堆旁边的柴火堆）
昭君走过去，拿起一根枯树枝。
她蹲下来，把树枝放进火堆。
火苗窜了一下，然后稳稳地烧起来。
【音效】 柴火噼啪声变大
【视觉】 火焰颜色：橙红 → 稳定

昭君（独白）：
"火小的时候，人也会觉得冷。"
"添一根柴，它就旺了。"
"人也一样吧。"

【互动点3：点击雨夜山林】
（玩家点击远处的山林）
她看着那片墨绿色的山林。
雨中的树，颜色很深，几乎发黑。
偶尔有一只乌鸦叫了一声，从树梢飞过去。
没有声音——至少画面里听不见。
但她好像听见了。

昭君（独白）：
"雨停了以后，天就会晴吗？"

【叙事文本】
她坐在火堆旁边，没有睡。
火光照着她的脸，一半亮一半暗。
外面是雨声，很远。
这里离长安有多远了？
她算了算。
大概……已经走了两天的路了。

【收藏品触发·山树叶】
（火光摇曳，一片山树叶从树上飘落）
叶子落在地上，叶脉在火光里清晰可见。
玩家点击树叶 → 收集"一片山树叶"
（后台：collectibles.qin_forest_leaf.unlocked = true）

【音效变化】
雨声里多了一个声音——不是雨。
是远处的什么动物在叫。
不像狼。更像……她自己不太确定。
也许是狐狸。
马夫在篷子另一边已经睡着了，鼾声轻微。

【结尾·夜深】
昭君裹着衣服，闭上眼睛。
没有睡着。
火堆噼啪响。
雨还在下。
【音效】 雨声渐弱，过渡到下一章的背景音

【章节结束】
镜头从火堆慢慢拉远。
山林在雨夜里变成一片墨绿色的海。
【字幕】
第三章·秦岭  完

【界面】
▶ 继续前行（进入第四章·黄河）
🎵 查看旅行册
```

## 设计亮点

1. **触觉意象**：车窗上的雨痕（手指跟随），添柴（火焰），触摸山林——三个互动都是"触觉"，贯穿本章
2. **火光的象征**：火堆是本章的核心意象——"火小的时候人也会觉得冷，添一根柴它就旺了"，既是物理动作，也是心理投射：**人在困境中的坚韧**
3. **闪回处理**：雨中发呆——极短（2秒），只有雨丝、织布机声、空无一人的房间。闪回之后立刻回到现实，手法极淡
4. **是否定的叙事**：本章没有"继续前行"按钮，只有"歇一晚"——山路难行，时间静止
5. **声音层次**：雨打树叶 → 雨打玻璃 → 闪回中的织布声 → 雨打篷顶 → 柴火噼啪 → 远处的未知声音——声音在推进剧情
