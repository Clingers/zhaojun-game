# 第二章：渭水

> **状态**：✅ 定稿（老板审阅通过，2026年7月）
> 
> **主题**：第一次离开故乡。怅惘、回望
> 
> **色调**：土褐 + 灰蓝
> 
> **音乐**：流水声（渭河） + 秋风 + 箫独奏
> 
> **时长**：约 2-3 分钟
> 
> **收藏品**：圆润的石头

---

## 完整剧本

```
【开场】 淡入
土黄色的原野。深秋的草已经枯了，风一吹就倒。
一条河在远处闪亮——渭河。
马车在路上行驶，很慢。
【音效】 流水声（轻微，远处） + 风声（不响）

【镜头1】 车内
昭君坐在马车里，看着窗外。
景色和长安附近的山不一样了。
更平，更远，更单调。
【音乐起】 箫独奏，缓慢，略带忧伤

【叙事文本·淡入】
车行了两个时辰。
她一直没有说话。
窗外的景色渐渐变了——宫殿不见了，山也不见了。
只剩下一片接一片的黄土。
还有那条河，一直跟在路边。

【互动点1：第一次回头看】
（玩家点击车窗玻璃触发）
马车帘子掀开。
昭君回望——长安方向。
远处的城墙已经看不见了，只剩下一片淡青色的雾，挂在天的尽头。
【音效】 风声忽然变大了一点点

昭君（独白）：
"已经看不见了。"

【互动点2：第二次回头看】
（玩家再次点击车窗触发）
她再望。
那团雾也淡了。
【音效】 风声持续

昭君（独白）：
"连城楼都看不见了。"

【互动点3：第三次回头看】
（玩家第三次点击车窗触发）
她还在看。
天尽头什么都没有了，只有平线。
【音效】 风声里多了一点呜咽感（箫的音色变化）

昭君（独白，更轻）：
"什么都没有了。"

【互动点4：无限次点击】
（玩家第四次及以后点击车窗）
昭君还在看。
但什么都不会再出现。
只有风，只有远处的渭河。
（此状态可重复，无新台词）

【叙事文本】
风吹进来，有点冷。
她把帘子放下。
手指无意识地在膝上敲了一下，像是想把什么东西按回去。
没有成功。

【中场：停车】
马车停了。
马夫的声音（车外，隔着一层帘子）：
"姑娘，前面就是渭河了。"
"要停一停吗？"

昭君：
"嗯。"

【互动点5：下车走到河边】
（玩家点击"下车看看"）
昭君踩到地面。
泥土是软的，秋天干燥后的那种。
她走到河边。
水是浑的，黄褐色，流速不快，但看得出一直在走。
【音效】 流水声变大，清晰

昭君（独白）：
"我小时候，家门口也有一条河——香溪。"
"夏天的时候，我们坐在石头上洗脚。"
"水很清，能看见水底的沙。"

【互动点6：点击水面】
（玩家点击河面，或停留超过3秒）
昭君蹲下来，用手捧了一点水。
水从指缝里流走。
她低头看着，看见指缝间漏下的光。
她忽然注意到脚边有一块石头——圆润的，被水冲了很久。
【音效】 细微的水声

【收集触发】
玩家点击石头 → 收集"圆润的石头"
（后台：collectibles.wei_stone.unlocked = true）

【叙事文本】
她站起来，拍了拍手上的水。
没有擦干——反正很快就会干的。
风吹过来，她裹了裹衣服。

【结尾·继续上路】
回到马车。
马车重新启动。
车轮压在土路上，吱呀作响。
【音效】 马蹄声渐强

昭君（独白）：
"过了河，就真的离开家了。"

【镜头】 马车过桥
渭河在车下流过。
水面上的光碎成一片，晃了一下。
【音乐变化】 箫声渐弱，过渡到下一章的音乐

【章节结束】
马车在黄土路上渐行渐远。
【字幕】
第二章·渭水  完

【界面】
▶ 继续前行（进入第三章·秦岭）
🎵 查看旅行册
```
