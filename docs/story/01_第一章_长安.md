# 第一章：长安

> **状态**：✅ 定稿（老板审阅通过，2026年7月）
> 
> **章节定位**：玩家进入的第一个叙事章节
> 
> **主题**：离别
> 
> **色调**：暖黄色（秋光、朱红、铜绿）
> 
> **音乐**：宫廷丝竹（远处） + 古筝慢板（淡入）
> 
> **时长**：约 2-3 分钟
> 
> **收藏品**：铜镜碎片

---

## 完整剧本

```
【开场】 淡入
深秋的清晨。光线从窗棂斜射进来，照亮空气中的微尘。
房间：未央宫·长门馆附近的一间小院。简单，安静，有岁月感。

【镜头1】 特写：铜镜
铜镜放在木架上。镜面已有划痕，但被擦拭得很干净。
有人刚刚擦过它。手离开镜面时留下一个模糊的指印。
【音效】 细微的丝绸摩擦声（有人在整理衣物）

【镜头2】 拉远
昭君背对镜头，坐在榻边。面前打开一只樟木行囊。
她穿深衣，素色，头发松松挽着，未施粉黛。
她拿起一件东西，看了很久，然后放进行囊。
【音效】 衣料摩擦声

【叙事文本·淡入】
她在收拾行李。
不是很多。五年了，她的东西其实不多。
一套换洗衣物，一面铜镜，一把琵琶。
还有几件别人送的零碎小物件。
够带走的，就这些。

【互动点·行囊】
（玩家点击行囊）
昭君拿起一件深衣，叠得更仔细了些。
她又拿起琵琶，指尖在弦上轻轻一拨。
没有声音——至少画面里听不见。
但她好像听见了什么。

昭君（独白·轻）：
"这琵琶是母亲给的。"
"她说，宫里日子长，要有件东西陪着我。"
"五年了，它还在。"

【互动点·铜镜】
（玩家点击铜镜）
昭君的手伸向镜架。
手指停在镜面上，犹豫了一下。
然后她移开了。

昭君（独白）：
"这镜子进宫那天我就带着。"
"五年，它见过我最多的样子。"
"大概……是我唯一的朋友。"

【收集品触发·铜镜碎片】
铜镜表面泛起一圈微弱的光。
一个小碎片从镜面脱落，飞向屏幕右上角。
画面角落出现背包图标闪烁一下。
（后台：collectibles.bronze_mirror.unlocked = true）

【互动点·发钗】
（玩家点击梳妆盒最上层）
昭君打开盒子，取出发钗。
银质，简简单单一个如意头。不是贵重东西。
她看了很久。

昭君（独白·更轻）：
"这是母亲在我及笄那年送的。"
"那天她给我梳头，说："
"'女儿长大了，要嫁人了。'"
"我当时笑她，说我才不嫁。"
"现在……"
"（停顿）"
"年少时的话，不作数的。"

【叙事文本】
她把发钗小心包好，放进行囊。
手指触到行囊底部的硬纸——是出行的路引。
她拿出来看了看，又放回去。
外面有脚步声。很轻，但很急。

【互动点·侍女对话】
（侍女出现在门口）
侍女（轻声）：
"姑娘，车队已经在宫门等候了。"
"汉使大人说……今日必须出城。"

昭君（没有回头）：
"知道了。"

侍女（顿了顿）：
"姑娘，您真的要走吗？"
"我……"

昭君（打断，但仍然温柔）：
"嗯。"

【叙事文本】
侍女没有再说什么，退了出去。
脚步声远了。
房间里又只剩她一个人。

【音乐变化】
远处的丝竹声停了。
古筝声起，缓慢，压抑的离别。

【中场·过场动画】
昭君站起身，最后看了一眼这间屋子。
铜镜、琵琶、空了的梳妆台。
她把行囊的带子系好，背上。
走到门口，手放在门框上。
停了三秒。

【闪回·极短】
（画面一闪）
深宫的走廊。一个女子的背影，越走越远。
不是昭君——是另一个宫女。
宦官的声音（画外）：
"下一个。"
脚步声，门关上。
画面回到现实。

【结尾·出宫门】
昭君走出院子。
院外是一条长长的夹道。
尽头是宫门——高，厚，沉。
几个宦官和宫女站在路边，目送。
没有人说话。

【镜头切换·元帝视角】
（闪回）
汉元帝站在大殿的台阶上，远眺宫门方向。
身边的内侍在说什么，他仿佛没听见。
他的手微微握了一下，又松开。

内侍（小心翼翼）：
"陛下？"

元帝（收回目光）：
"朕没事。"
"她走了？"

内侍：
"是。车队已出城。"

元帝（沉默片刻）：
"知道了。"

【镜头回到昭君】
马车已经在宫门外等候。
昭君上车。
帘子放下来。
车轮滚动。

【镜头·车内特写】
她的手放在膝上。
行囊就在脚边。
她向外看了一眼——宫门在视野里越来越小。

【叙事文本·淡入】
她没有回头。
长安的城楼在秋阳里沉成一片淡金色。
很远，很安静。

【章节结束】
马车转过街角，宫门消失在视线里。
【音乐淡出】
【黑屏】
【字幕浮现】
第一章·长安  完

```
