# 第六章：草原

> **状态**：✅ 定稿（老板审阅通过，2026年7月）
> 
> **主题**：进入匈奴领地。天空巨大，画风突变
> 
> **色调**：钴蓝 + 草绿（颜色从上一章的灰黄骤变）
> 
> **音乐**：马头琴（悠远，辽阔）
> 
> **时长**：约 2-3 分钟
> 
> **收藏品**：白鹰羽

---

## 完整剧本

```
【开场】 淡入
不是渐变——是**切换**。
上一秒还是雁门关的灰黄，这一秒忽然变成一片蓝绿色。
天空是钴蓝色的，高得吓人。
地面是草绿的，一望无际。
没有山，没有树，只有草，和风。
【音效】 风声变了——不再是山壁间的呼啸，而是草原上的开阔风声
【音乐起】 马头琴，悠远，辽阔，像从很远的地方飘过来

【镜头1】 马车在草原上
路不是路了，只是草比较低，车辙印还在。
远处有几只羊，在吃草。
再远处，有一个小黑点——是个骑马的人。

【叙事文本·淡入】
过了雁门关，天地忽然就大了。
不是一点点变大，是**突然**变大。
山没有了。树没有了。
只剩天空，和草。
还有风——草原上的风，和关里的风不一样。

【互动点1：第一次看见羊群】
（玩家点击羊群）
昭君从来没有见过这么多羊。
它们不是白色的，是浅棕色的，带着点黄。
一只母羊带着几只小羊，在吃草。
看见马车过来，它们抬头看了一眼，然后继续吃。
【音效】 羊的轻微咩叫（偶尔）

昭君（独白）：
"它们在吃什么？"
"草有这么好吗？"

【互动点2：第一次看见牧民】
（玩家点击远处的小黑点）
那个小黑点近了一点——是一个匈奴牧民，骑着马。
他穿着皮衣，头上戴着皮帽。
看见马车，他挥了挥手。
昭君也挥了挥手。
他笑了一下，然后策马走远了。
【音效】 马蹄声（由远及近又远去）

昭君（独白）：
"他们住在这里？"
"冬天怎么办？"

【互动点3：第一次看见鹰】
（玩家点击天空中的一个小点）
一只鹰在高空盘旋。
翅膀很大，展开的时候几乎遮住了天空的一角。
它不叫，只是飞。
一圈，一圈，越来越高。
【音效】 鹰的啼声（高亢，尖利，划过天空）

昭君（独白）：
"我在宫里的时候，有人在墙上画过这种鸟。"
"他们叫它……什么来着？"
"记不得了。"

【闪回·极短】
（画面一闪）
深宫的墙角落。
一幅画，画着一只展翅的鸟。
旁边题字："大鹏一日同风起"。
小昭君蹲在墙边看。
宫女的喊声（远处）：
"王昭君——你又跑哪里去了！"
【闪回结束】
【镜头回到草原】
鹰还在飞。

【互动点4：看自己的衣服】
（玩家点击昭君的衣襟）
她低头看了看自己的汉服——深衣，丝织品，颜色素净。
和周围的一切都不一样。
这里的草是绿的，天是蓝的，她的衣服是……另一种颜色。
【音效】 风声轻了一下

昭君（独白）：
"他们一直在看我。"
"大概是因为我的衣服不一样。"

【叙事文本】
她忽然想起母亲教她绣花的样子。
针线穿过布帛，一朵白梅花慢慢浮现。
"女子要学会针线，到哪里都能有一口饭吃。"
当时她不懂。
现在她忽然有点懂了。

【收藏品触发·白鹰羽】
（鹰在高空盘旋5秒，一片羽毛飘落）
羽毛在风里飘了很久，终于落在马车边。
玩家点击羽毛 → 收集"白鹰羽"
（后台：collectibles.eagle_feather.unlocked = true）

【结尾·继续前行】
马夫的声音：
"姑娘，我们得快一点了。"
"天黑前要赶到下一个营地。"

昭君：
"嗯。"

马车加速。
草原在视野里展开，越来越大。
天空越来越低。
【音乐】 马头琴声渐弱，过渡到下一章的音乐

【章节结束】
【字幕】
第六章·草原  完

【界面】
▶ 继续前行（进入第七章·暴风雪）
🎵 查看旅行册
```

## 设计亮点

1. **画风突变**：本章开场是"切换"而非"渐变"，让玩家直观感受到"进入了另一个世界"
2. **天空巨大**：草原的核心意象——天空占画面的2/3，地面只有1/3，产生渺小感
3. **三次"第一次"**：第一次看见羊群、第一次看见牧民、第一次看见鹰——都是她从未见过的草原生活
4. **鹰的意象**：呼应序章的大雁，但鹰更高、更孤独、更有力量——草原的象征
5. **汉服的对比**：她忽然注意到自己的衣服和周围格格不入——"他们一直在看我"，暗示她的"他者"身份
6. **母亲教绣花的闪回**：极短，只有"女子要学会针线，到哪里都能有一口饭吃"——这是她日后在匈奴传播纺织技术的伏笔
7. **白鹰羽的命名**：采用古典文学中对鹰羽的常见表述方式，简洁有古意
