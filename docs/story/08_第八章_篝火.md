# 第八章：篝火

> **状态**：✅ 定稿（老板审阅通过，2026年7月）
> 
> **主题**：夜晚。围坐，匈奴老人唱歌
> 
> **色调**：暖橙 + 深褐（与第七章冷白形成强烈对比）
> 
> **音乐**：篝火噼啪声 + 虫鸣 + 匈奴老人古歌（无歌词古民谣，可循环）
> 
> **时长**：约 2-3 分钟
> 
> **收藏品**：篝火炭

---

## 完整剧本

```
【开场】 淡入
不是渐变——是**切换**。
上一秒还是暴风雪的惨白，这一秒忽然变成暖橙色。
黑夜。草原的夜空，星星很密。
一团篝火在画面中央，噼啪响。
几个人围坐在火堆旁边——马夫、两个匈奴人、昭君。
火光照着他们的脸，一半亮一半暗。
【音效】 篝火噼啪声（持续） + 虫鸣声（秋夜，轻微） + 风声（但很弱，被篝火挡住了）

【音乐起】 篝火背景音（持续） + 古歌（渐入）

【镜头1】 火堆
火焰在跳。木柴爆开一朵小火花。
昭君坐在一边，抱着膝盖。
她还没有完全从风雪里缓过来，但已经好多了。
【音效】 呼吸声（平缓了）

【叙事文本·淡入】
匈奴人找到了他们。
带他们到这个篝火旁边。
火很旺。很暖。
昭君把手伸向火，手指碰到热气的边缘。

【互动点1：看火】
（玩家点击篝火）
火焰跳了一下。
她看着火，眼睛被火光映成橙色。
很远的地方，风还在吹。
但这里很安静。
【音效】 柴火噼啪声变大，然后恢复

【叙事文本】
一个匈奴老人坐在另一边。
他一直在看昭君，看了很久。
然后他开口了——不是说话，是唱歌。
【音效变化】 古歌起（人声，无歌词，苍老，悠远，类似古民谣）

【古歌】
（无歌词。只有旋律。像风，像草原，像很久以前的事。）
老人唱得很慢，声音沙哑，但很稳。
每一句都拖得很长，像草原上的地平线。
他唱的时候，其他人都不说话了。
连火堆都好像安静了一点。
【音效】 古歌持续，循环播放（可以随时停止）

【互动点2：听歌】
（玩家点击"听歌"按钮，或点击老人）
昭君转过头，看着老人。
她听不懂歌词——这首歌本来就没有歌词。
但她能听懂那种感觉。
那是草原上的人，唱给草原听的声音。
【音效】 古歌音量微微加强，但保持远处感

【互动点3：加柴】
（玩家点击柴火堆）
昭君拿起一根枯树枝，放进篝火。
火苗窜了一下，照亮了她的脸。
她笑了一下——这是她出宫以来第一次笑。
【音效】 柴火噼啪声变大

昭君（独白，极轻）：
"火很暖。"

【叙事文本】
她忽然想起母亲教她绣花的时候，也是这样的夜晚。
窗外有风。灯很暗。
母亲说："你看，针线要慢慢来。"
火堆的光，和那时候的灯，好像。
但又不完全一样。
这里的火更亮，更跳，更……自由。

【互动点4：看星星】
（玩家点击天空）
她抬头看星星。
草原的星星比长安密得多。
一颗，两颗，数不清。
她在宫里的时候，也看过星星。
但从没有这样密集过。
【音效】 风声很弱，虫鸣声轻微

昭君（独白）：
"它们一直在那里吗？"
"还是今晚才来的？"

【收藏品触发·篝火炭】
（火堆里掉下一块烧过的炭，暗红色）
炭块在地上滚了一下，停住。
玩家点击炭块 → 收集"篝火炭"
（后台：collectibles.campfire_charcoal.unlocked = true）

【结尾·夜深】
古歌还在唱。
但老人慢慢合上了眼睛。
其他人也开始打瞌睡。
火堆噼啪响，声音越来越慢。
昭君也闭上了眼睛。
这次她睡着了。

【章节结束】
镜头从篝火拉远——夜色、草原、星星、火堆的光。
渐渐暗下去。
【字幕】
第八章·篝火  完

【界面】
▶ 继续前行（进入第九章·王庭）
🎵 查看旅行册
```

## 设计亮点

1. **暖橙对冷白**：本章是色彩对比最强的一章——上一章是暴风雪的惨白，这一章是篝火的暖橙。玩家打开第八章的瞬间就能感受到"安全了"
2. **古歌的设计**：没有歌词的古民谣——让音乐成为环境音的一部分，而不是"插入的音乐"。老人唱得很慢，像地平线一样长。
3. **"可以一直听"的交互**：古歌循环播放，玩家不强制推进，可以停下来听完整首歌再走。这是本章的核心体验。
4. **第一次笑**：她出宫以来第一次笑——在加柴的时候，火光映脸。微小但重要的瞬间。
5. **篝火的象征**：从"帐篷"（第七章）到"篝火"（第八章）——从躲避风雪到主动取暖，从生存到生活。她开始适应。
