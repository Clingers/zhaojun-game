# 角色设定

> 《昭君：一路向北》
> 版本：V1.0 | 2026年7月

---

## 昭君

| 项目 | 内容 |
|------|------|
| **姓名** | 王嫱，字昭君（"昭"为封号） |
| **年龄** | 出塞时约20岁 |
| **籍贯** | 南郡秭归（今湖北兴山） |
| **性格** | 安静、内敛、有主见、不轻易表露情感 |
| **关键特质** | 沉默。不是无话可说，而是选择不说 |
| **声音** | 平静、略带感伤的年轻女声，语速慢 |
| **外貌** | 清瘦，面容清秀，不艳丽。眼神沉静 |
| **语言风格** | 现代散文风，口语但不随意，偶尔带诗意 |
| **象征物** | 琵琶、铜镜 |

### 台词语感示例

- "长安城的钟声，今天格外响。"
- "我在宫里待了五年。五年。"
- "出塞……也不是件坏事。"

## 角色形象原则

- **昭君**是唯一有详细外貌描述的角色
- 其他角色以声音+台词塑造，不依赖视觉形象
- 昭君的形象随章节变化（从深宫到草原），但气质不变
