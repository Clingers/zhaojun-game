## 完整剧本

```
【镜头1】 黑屏
【音效】 风声（高空，呼啸） + 远处钟声（闷响，3秒一次）

【镜头2】 淡入
深秋天空，澄澈，苍青
一行大雁（V字）向南飞
镜头缓慢推进，跟随其中一只
【音效】 雁鸣（偶尔）

【镜头3】
那只大雁忽然停住，翅膀悬空不动
【镜头4】 俯拍
地面：长安城墙如灰青色长线
城门裂开一个口子
车队驶出（马车 × 3，随从 × 4，无旌旗）
【音效】 马蹄声（渐入又渐弱）

【镜头5】
大雁低头看
车队中间马车的帘子掀开
昭君的脸探出——年轻，清瘦，平静
她回望长安（城楼在秋阳里泛淡金色）
不说话，只是看着
【停顿】 5秒（只有风声）

【镜头6】
昭君放下帘子
大雁空中转半圈——方向改变，不再向南，而是跟随车队

【镜头7】 拉远
天地苍茫，大雁在车队上空盘旋
路伸向北方，消失在远山之后
【字幕】 昭君：一路向北

【镜头8】 全黑
【停顿】 2秒

【界面】
▶ 开始旅程
🎵 声音设置
```

## 美术参考

| 元素 | 风格参考 |
|------|----------|
| 天空 | 北宋郭熙《早春图》——苍青，高远 |
| 大雁 | 工笔，但只画轮廓，不填细节 |
| 长安城墙 | 淡墨渲染，如隔雾观花 |
| 车队 | 简笔，灰调，不突出 |
| 昭君 | 背影/侧影，红衣一点（朱砂），不露全貌 |

## 音频 cue 表

| 时间点 | 音效/音乐 |
|--------|----------|
| 0:00 | 风声起（淡入2秒） |
| 0:05 | 远处钟声第一声 |
| 0:20 | 雁鸣（1次） |
| 0:30 | 马蹄声（车队出现） |
| 0:45 | 马蹄声减弱 |
| 1:00 | 风声持续（无音乐） |

---

## 设计决策

- 无对白，纯视觉 + 环境音
- 钟声"3秒一次"→ 象征长安告别
- 大雁不鸣叫，只有展翅的寂静
- 昭君台词：无。她的沉默就是台词
