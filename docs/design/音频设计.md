# 音频设计文档

> 《昭君：一路向北》
> 版本：V1.0 | 2026年7月

---

## 核心原则

**声音比画面重要。**

整个游戏以环境音为主，配乐为辅。各章节的音景要能单独循环，让玩家沉浸其中。

---

## 音频层级

```
环境音（ambient） ← 底层循环，持续整个章节
   ↓
配乐（music）     ← 中层，淡入淡出切换
   ↓
配音（voice）     ← 顶层，对话/独白
   ↓
SFX              ← 交互反馈
```

## 配音（TTS预生成）

| 内容 | 建议声音 |
|------|----------|
| 昭君独白 | 年轻女声，平静，略带感伤 |
| 昭君对话 | 同上，情绪略波动 |
| 汉使 | 中年男声，沉稳 |
| 侍女 | 年轻女声，轻柔 |
| 马夫 | 中年男声，粗犷 |
| 匈奴老人 | 老年男声，苍劲 |

**技术**：使用 Azure TTS 或 ElevenLabs 预生成 MP3，存入 `assets/audio/voice/`，运行时直接播放。

## 技术实现

```typescript
// src/systems/AudioManager.ts
class AudioManager {
  private ctx: AudioContext;
  private loops: Map<string, { el: HTMLAudioElement; gain: GainNode }>;
  
  // 环境音循环（无缝loop）
  playAmbient(key: string, fadeMs = 2000): void;
  
  // 章节音乐切换（淡入淡出）
  switchMusic(key: string, fadeIn = 3000, fadeOut = 2000): void;
  
  // 配音播放（排队）
  playVoiceQueue(lines: DialogueLine[]): Promise<void>;
  
  // 全局音量
  setVolume(type: 'ambient'|'music'|'voice', value: number): void;
}
```
