## 二、详细修复内容

### P0-01：React Hook 在 Phaser 类中调用

#### 问题描述
`BaseScene` 中直接调用 `useCollectiblesStore.getState()` 和 `useGameStore.getState()`，这是 React Hook，在 Phaser 类中调用会导致运行时错误。

#### 错误代码（BaseScene.ts:86-90）
```typescript
protected onCollectibleCollected(collectibleId: string) {
  const { unlock } = useCollectiblesStore.getState(); // ❌ 错误！
  unlock(collectibleId);
  this.showDialogue('收集成功');
}
```

#### 修复方案
改为**事件回调**模式：
- `BaseScene` 不再直接调用 Zustand store
- 改为触发自定义事件：`dialogue`、`collect`、`transition`
- React 侧监听这些事件并更新 store

#### 修复后代码
```typescript
export abstract class BaseScene extends Phaser.Scene {
  private dialogueCallback?: (event: DialogueEvent) => void;
  private collectCallback?: (event: CollectEvent) => void;
  private transitionCallback?: (event: TransitionEvent) => void;

  constructor(chapterId: string, audioManager: AudioManager) {
    super({ key: chapterId });
    this.chapterId = chapterId;
    this.audioManager = audioManager;
  }

  setEventHandlers(
    onDialogue: (event: DialogueEvent) => void,
    onCollect: (event: CollectEvent) => void,
    onTransition: (event: TransitionEvent) => void
  ) {
    this.dialogueCallback = onDialogue;
    this.collectCallback = onCollect;
    this.transitionCallback = onTransition;
  }

  protected onCollectibleCollected(collectibleId: string, item: any) {
    if (this.collectCallback) {
      this.collectCallback({
        type: 'collectible',
        collectibleId,
        item,
      });
    }
    this.showDialogue(`收集成功：${item.name}`);
  }
}
```

#### 修复文件
- `src/scenes/BaseScene.ts`
- `src/scenes/PrologueScene.ts`
- `src/scenes/Chapter01Scene.ts` ~ `Chapter10Scene.ts`

### P0-02：全局变量 `window.gameEngine` 耦合

#### 问题描述
`BaseScene.create()` 中通过 `(window as any).gameEngine` 查找 `AudioManager`，强依赖全局变量，耦合度高且难以测试。

#### 错误代码（BaseScene.ts:24-29）
```typescript
create() {
  const gameEngine = (window as any).gameEngine as any; // ❌ 全局变量
  if (gameEngine) {
    this.audioManager = gameEngine.getAudioManager();
  }
  // ...
}
```

#### 修复方案
**依赖注入**：
- `BaseScene` 构造函数增加 `audioManager: AudioManager` 参数
- `GameEngine` 创建场景时注入 `AudioManager` 实例
- `SceneManager` 持有 `AudioManager` 引用，创建场景时传入

#### 修复后代码

**SceneManager.ts**
```typescript
export class SceneManager {
  private audioManager: AudioManager;

  constructor(audioManager: AudioManager) {
    this.audioManager = audioManager;
  }

  getScenes(): Phaser.Scene[] {
    return this.sceneConfigs.map(
      (config) => new config.sceneClass(this.audioManager)  // ✅ 注入
    );
  }
}
```

**BaseScene.ts**
```typescript
export abstract class BaseScene extends Phaser.Scene {
  protected audioManager: AudioManager; // ✅ 不再需要 setAudioManager

  constructor(chapterId: string, audioManager: AudioManager) {
    super({ key: chapterId });
    this.audioManager = audioManager; // ✅ 构造函数注入
  }
}
```

**GameEngine.ts**（已移除全局变量引用，仅保留用于调试的可选引用）

#### 修复文件
- `src/core/GameEngine.ts`
- `src/core/SceneManager.ts`
- `src/scenes/BaseScene.ts`
- `src/scenes/PrologueScene.ts` ~ `Chapter10Scene.ts`

### P0-03：类型定义缺失/未导出

#### 问题描述
项目中缺少统一的类型定义文件，`GameSettings`、`Collectible` 等类型未导出，导致类型安全失效。

#### 修复方案

**1. 创建 `src/types/index.ts`**

```typescript
// 游戏配置相关类型
export interface ChapterConfig { /* ... */ }
export interface SceneConfig { /* ... */ }
export interface CollectibleConfig { /* ... */ }

// 音频相关类型
export interface AudioConfig { /* ... */ }

// 存档相关类型
export interface SaveData {
  saveVersion: string; // ✅ 添加版本号
  currentChapter: string;
  progress: Record<string, boolean>;
  collectibles: string[];
  settings: GameSettings;
}

export interface GameSettings {
  volume: number;
  subtitles: boolean;
}

// 事件相关
export interface DialogueEvent { type: 'dialogue'; text: string; duration?: number; }
export interface CollectEvent { type: 'collectible'; collectibleId: string; item: any; }
export interface TransitionEvent { type: 'transition'; from: string; to: string; }
export type GameEvent = DialogueEvent | CollectEvent | TransitionEvent;
```

**2. 更新 `SaveManager.ts`**

```typescript
interface SaveData {
  saveVersion: string;  // ✅ 添加版本号
  currentChapter: string;
  progress: Record<string, boolean>;
  collectibles: string[];
  settings: GameSettings;
}

const SAVE_VERSION = '1.0';

save(data: Omit<SaveData, 'saveVersion'>): void {
  const saveData: SaveData = { ...data, saveVersion: SAVE_VERSION }; // ✅ 自动添加版本
  this.storage.setItem(SAVE_KEY, JSON.stringify(saveData));
}

load(): SaveData | null {
  const data = JSON.parse(saved) as SaveData;
  if (data.saveVersion !== SAVE_VERSION) { // ✅ 版本检查
    console.warn('Save version mismatch');
    return null;
  }
  return data;
}
```
