# 《昭君：一路向北》项目总览

> 交互式叙事游戏（Web互动叙事游戏）
> 让玩家"经历"昭君出塞，而不是"阅读"昭君出塞。

## 📋 基本信息

| 项目 | 内容 |
|------|------|
| **项目名称** | 《昭君：一路向北》 |
| **类型** | Web互动叙事游戏（Narrative Adventure） |
| **关键词** | AIGC、历史文化、交互叙事、声音设计、轻游戏、人文体验 |
| **游玩时长** | 20~30分钟 |
| **目标** | 让玩家"经历"昭君出塞，而不是"阅读"昭君出塞 |

## 🎯 核心理念

整个游戏没有传统意义上的：
- 血量
- 战斗
- 升级
- 装备

玩家唯一需要做的是：**陪昭君走完这一千多公里。**

玩家不是昭君。玩家更像：
- 一阵风
- 一位同行者
- 一个见证者

因此整个游戏十分安静。

## 📖 章节结构

```
开始游戏 → 卷轴展开（序章）→ 长安 → 渭水 → 秦岭
→ 黄河 → 雁门关 → 草原 → 暴风雪 → 篝火
→ 王庭 → 终章
```

共10个章节，每章约2~3分钟。

## 🎮 四种操作

1. **前进** — 点击"继续前行"，马车开始移动
2. **停留** — 停下来观察，点击场景元素触发文本
3. **对话** — 与汉使、侍女、马夫、匈奴使者聊天
4. **收藏** — 收集风、声音、诗句、花、石头、地图印章

形成《一路向北旅行册》

## 🛠 技术方案

| 模块 | 技术 |
|------|------|
| 前端 | React + Vite |

| 配音 | AI TTS预生成 |
| 部署 | GitHub Pages 或 Vercel |

## 💡 创新亮点：记忆系统

收集"记忆碎片"：一阵秦岭的雨声、一句离别的话语、一片草原的落叶、一首胡笳曲。抵达单于庭后自动组合成专属《一路向北》电子画册。每次游玩的画册内容略有差异。
