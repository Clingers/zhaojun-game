## 技术栈总览

| 层级 | 技术选型 | 理由 |
|------|----------|------|
| **框架** | React 18 + TypeScript | 组件化开发，生态成熟 |
| **构建工具** | Vite 5 | 极速开发体验，原生ESM支持 |
| **游戏引擎** | Phaser 3 | 2D场景管理、交互、动画成熟 |
| **动画库** | GSAP 3 | 时间轴控制，复杂动画编排 |
| **音频** | Web Audio API + Howler.js | 环境音循环、淡入淡出、多轨道混音 |
| **状态管理** | Zustand | 轻量，适合管理游戏状态（进度/收藏品） |
| **路由** | React Router v6 | 章节切换、设置页面 |
| **CSS** | Tailwind CSS + CSS Modules | 快速布局 + 章节专属样式隔离 |
| **部署** | Vercel / GitHub Pages | 零配置部署 |

## 核心系统设计

### 1. 音频系统 (AudioManager)

**职责**：
- 环境音循环播放（风声、雨声、宫廷丝竹）
- 章节配乐淡入淡出切换
- 对话配音播放
- SFX 触发（点击音、章节切换音）
- 全局音量控制

**实现思路**：
```typescript
// src/systems/AudioManager.ts
class AudioManager {
  private audioContext: AudioContext;
  private loops: Map<string, HTMLAudioElement>;
  
  playAmbient(key: string, fadeIn?: number);
  playMusic(key: string, fadeIn?: number, fadeOut?: number);
  playVoice(lineId: string);
  playSFX(key: string);
  setVolume(type: 'ambient' | 'music' | 'voice' | 'sfx', value: number);
}
```

**资源规范**：
- 环境音：30-60秒循环片段，-3dB 音量
- 章节配乐：2-3分钟，淡入淡出
- 配音：MP3 128kbps，TTS预生成

---

### 2. 记忆/收藏系统 (MemorySystem)

**职责**：
- 追踪已收集的记忆碎片
- 计算当前章节可收集物品
- 触发"发现新收藏品"动画
- 保存进度到 localStorage
- 合成《一路向北旅行册》

**数据结构**：
```typescript
interface Collectible {
  id: string;
  name: string;
  description: string;
  chapterId: string;
  icon: string;              // 图标路径
  unlocked: boolean;
  discoveredAt?: Date;
}

interface InventoryState {
  collectibles: Collectible[];
  totalCount: number;
  chaptersCompleted: string[];
}
```

---

### 3. 对话系统 (DialogueManager)

**职责**：
- 管理角色对话队列
- 打字机效果逐字显示
- 对话头像/角色名显示
- 选项分支（如有）

**实现**：
```typescript
interface DialogueLine {
  speaker: string;           // "昭君" | "汉使" | "侍女" | "马夫"
  text: string;
  voice?: string;            // TTS音频路径
  emotion?: 'neutral' | 'sad' | 'wistful';
}
```

---

### 4. 章节场景系统 (Scene System)

**职责**：
- 每章独立组件，按路由切换
- 场景内部状态管理（当前段落、交互进度）
- 与 AudioManager 同步切换环境音
- GSAP 动画编排（入场、过渡、离开）
- 触发 MemorySystem 收集逻辑

**章节组件模板**：
```tsx
// src/scenes/Chapter1_ChangAn.tsx
export default function Chapter1_ChangAn() {
  const { currentSegment, advance } = useChapterProgress('chapter1');
  const { addCollectible } = useInventory();
  const audio = useAudio();
  
  useEffect(() => {
    audio.playAmbient('palace_bells');
    audio.playMusic('chapter1_music');
  }, []);
  
  return (
    <ChapterContainer>
      <NarrationText text={currentSegment.text} />
      {currentSegment.type === 'interaction' && (
        <InteractiveHotspot onClick={advance} />
      )}
    </ChapterContainer>
  );
}
```

## 性能与优化

| 方面 | 策略 |
|------|------|
| **图片** | WebP 格式，懒加载章节背景 |
| **音频** | 预加载当前+下一章，卸载已完成章节资源 |
| **代码分割** | React.lazy() 按章节懒加载 |
| **缓存** | Service Worker 缓存静态资源 |
| **首屏** | 序章先加载，其余章节流式加载 |
