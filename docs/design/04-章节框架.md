# 章节框架与讨论记录

> 游戏共10章 + 1序章 + 终章
> 每章约2~3分钟

---

## 🗺 结构总图

```
序章「一只大雁」
     ↓
第一章「长安」—— 离别，暖黄色，宫廷丝竹
     ↓
第二章「渭水」—— 第一次离开故乡，回头看
     ↓
第三章「秦岭」—— 下雨，添柴，山林
     ↓
第四章「黄河」—— 小游戏过河
     ↓
第五章「雁门关」—— 离开汉朝，城门关闭，关键转折
     ↓
第六章「草原」—— 画风突变，天空巨大
     ↓
第七章「暴风雪」—— 找避风处，画面变白
     ↓
第八章「篝火」—— 夜晚围坐，匈奴老人唱歌
     ↓
第九章「王庭」—— 抵达，安静，昭君下车
     ↓
终章「二十年后」
```

## 📝 各章讨论记录

### 序章 ✅ 已定稿
- 方向A（画轴自述体）→ 未采用
- 方向B（昭君独白）→ 未采用
- **方向C（大雁眼睛）→ 定稿**

### 第一章：长安（待写）
史实点：
- 呼韩邪单于第三次来朝，"自请为婿"
- 昭君来自秭归，距长安千里，入宫后已与家人隔绝
- "昭君"很可能是封号——"昭，日明也"，代表汉皇光照匈奴

收藏品建议：铜镜碎片

### 第二章：渭水（待写）
史实点：
- 渭河是长安附近第一条大河
- 渡过渭河就是真正离开

设计亮点：每次点击"回头看"，长安越来越远，昭君一句话
收藏品建议：渭河的一捧水 / 长安的最后一瞥

### 第三章：秦岭（待写）
史实点：
- 翻越秦岭，可能融合风陵渡渡黄河
- 在黄河边最后一次回望南方

收藏品建议：一片栌叶 / 雨声记忆

### 第四章：黄河（待写）
小游戏：帮助车队过河，不用失败只是体验
音乐：鼓
收藏品建议：黄河的一粒沙

### 第五章：雁门关（待写）← 关键转折
史实点：
- 雁门关是汉匈界关，"塞"就在这里
- "雁池"传说——昭君经过时大雁因她的美丽坠落
- 出塞寓意："妹妹你大胆地往前走，莫回头"

设计要点：城门缓缓关闭，音乐停止，只剩风声
收藏品建议：关门上的土 / 一片雁羽

### 第六章：草原（待写）
史实点：
- 进入匈奴领地
- 昭君传播农业纺织技术的史实——牧民好奇看她的汉服刺绣

设计要点：天空巨大，颜色变成蓝色，音乐胡笳
收藏品建议：一根鹰羽 / 第一眼草原

### 第七章：暴风雪（待写）
史实点：漠北冬季严酷，"胡天八月即飞雪"
设计要点：风雪象征内心挣扎，呼应"上书求归"的伏笔
收藏品建议：一片雪花 / 风声记忆

### 第八章：篝火（待写）
史实点：匈奴歌谣传统、"胡笳"乐器
设计要点：匈奴老人唱歌，玩家可以一直听
收藏品建议：一段胡笳曲 / 一颗烧过的炭

### 第九章：王庭（待写）
史实点：单于庭在漠北（今蒙古国境内），不是呼和浩特
设计要点：安静，没有庆典，昭君下车
**记忆碎片在此处合成《一路向北旅行册》**

### 终章（待写）
史实点：她再也没有回到长安
设计要点：孩子奔跑，昭君站在远方
字幕："她再也没有回到长安。"
字幕后加一行："她死后葬于大黑河南岸，墓上青草四季常青，人称青冢。"
