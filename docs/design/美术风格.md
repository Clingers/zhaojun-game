# 美术风格指南

> 《昭君：一路向北》
> 版本：V1.0 | 2026年7月

---

## 总体风格

**国风水墨 × 半写实**，接近：
- 宋代长卷的平远构图
- 敦煌壁画的岩彩质感
- 工笔淡彩的细腻
- 水彩的渗透感

**非3D**，全部采用AI生成静态背景 + 手工后期。

## 背景图规范

- **分辨率**：1920×1080（16:9）为主，部分特写可用竖构图
- **格式**：WebP（优先）+ PNG 备用
- **风格**：写意水墨，避免过度细节导致喧宾夺主
- **人物处理**：背景不出现清晰人物，避免与UI文字争抢注意力

## UI 元素风格

### 按钮

```
继续前行 ──→ 毛笔墨字风格，衬线字体
            （边角轻微泛黄，如宣纸）
```

### 对话气泡

```
┌──────────────────┐
│ "……"            │ ← 宋体/楷体，浅色半透明背景
└──────────────────┘
```

### 收集品图标

- 铜镜碎片：圆形，铜锈色
- 一片枫叶：金红色
- 石头：灰蓝色
- 印章：朱红色方块
- 诗句：毛笔字条

---

## 动画风格

| 类型 | 实现 | 节奏 |
|------|------|------|
| 背景滚动 | GSAP slow pan | 30秒-1分钟完成一次循环 |
| 文字淡入 | GSAP fadeIn | 0.8秒 |
| 点击反馈 | 微微缩放 | 0.1秒 |
| 收集品飞入 | 贝塞尔曲线飞向右上 | 1.5秒 |
| 卷轴展开 | 从左到右展开 | 3秒 |

**所有动画以 `ease-out` 为主，禁用弹跳。**

---

## 字体

**正文**：Noto Serif SC（思源宋体）
**标题**：霞鹜文楷（LXGW WenKai）或 ZCOOL XiaoWei
**UI按钮**：与标题一致

**字号基准**：
- 叙事文本：18px / 行高 1.8
- 对话文本：16px
- 按钮文字：16px
- 标题：32-48px
