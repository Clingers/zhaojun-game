# 交互设计文档

> 《昭君：一路向北》
> 版本：V1.0 | 2026年7月

---

## 交互原则

1. **极简操作**：整个游戏只有四种操作，无复杂输入
2. **沉浸优先**：隐藏所有非必要UI，界面元素只在需要时出现
3. **慢节奏**：动画和文字节奏缓慢，留白充足
4. **触觉反馈**：所有可点击区域必须有视觉反馈

---

## 四种操作设计

### 1. 前进

**形式**：点击 "继续前行" 按钮
**效果**：
- 当前段落渐出
- 下一段淡入
- 马车开始移动（GSAP 动画）
- 背景图缓慢平移

**位置**：右下角，半透明，仅在叙事段落出现

---

### 2. 停留（观察）

**形式**：点击场景中的热区（hotspot）
**触发**：
- 玩家未点击前进，停留超过3秒 → 出现第一个热区提示
- 或显性展示可交互元素图标

**热区类型**：

| 元素 | 效果 | 收集品 |
|------|------|--------|
| 行囊 | 昭君拿起一件旧物，想起家乡 | 铜镜碎片 |
| 窗外风景 | 昭君出神凝望 | 一片远山 |
| 琵琶 | 手指轻拨琴弦，无声 | 琴弦记忆 |
| 侍女 | 触发对话 | — |
| 汉使 | 触发对话 | — |

**视觉反馈**：鼠标悬停时图标微微发光，点击后热区消失

---

### 3. 对话

**形式**：点击角色 → 弹出对话气泡 → 打字机效果逐字显示
**角色**：
- 汉使（送亲使臣）
- 侍女（贴身丫鬟）
- 马夫（车夫）
- 匈奴使者（后续章节）

**对话长度**：每段3-5句，散文风格，不要太密集
**退出**：点击屏幕空白处或"继续"关闭对话框

---

### 4. 收藏

**形式**：触发隐藏收集品 → 出现光点 → 点击确认 → 背包+1
**UI**：
- 发现时：物品图标从屏幕中央飞入右上角背包
- 背包入口：右上角小图标，点击弹出《一路向北旅行册》
- 旅行册：网格布局展示所有收集品，已解锁彩色，未解锁灰色

## 界面布局

### 游戏界面（标准模式）

```
┌──────────────────────────────────────────┐
│ [声音图标]  [菜单按钮]                    │  ← 顶部栏，极淡
├──────────────────────────────────────────┤
│                                          │
│           背景层（缓慢平移）               │
│                                          │
│         [叙事文本]  ← 居左/居中            │
│                                           │
│        [角色名]                           │
│        "对话内容……"                       │
│                                          │
│              [▶ 继续前行]  ← 右下         │
└──────────────────────────────────────────┘
```

### 旅行册（Album）

```
┌──────────────┐
│ 一路向北     │ ← 返回按钮
│ 旅行册       │
│ ─────────────│
│ [收集品1] [收集品2]  ← 网格，已解锁亮，未解锁灰
│ [收集品3] [收集品4]
│ ...          │
└──────────────┘
```

点击单个收集品 → 弹出详情弹窗（名称 + 描述文字）

## 章节过渡

| 过渡元素 | 效果 |
|---------|------|
| 章节开始 | 黑屏淡入 → 背景图缓慢展开（像卷轴） |
| 章节结束 | 画面渐隐 → 下一章标题浮现 → 点击进入 |
| 章节切换 | 音乐淡出淡入，环境音平滑过渡 |

---

## 进度系统

- 自动保存：每章结束后保存到 localStorage
- 章节解锁顺序：线性解锁（chapter1 → chapter2 → ...）
- 跳过已读：已完成的章节显示"重玩"选项
- 新游戏：重置所有进度

---

## 无障碍考虑

- 文字大小可调（设置中）
- 跳过对话动画（"全部显示"开关）
- 键盘支持：Space / Enter 继续
