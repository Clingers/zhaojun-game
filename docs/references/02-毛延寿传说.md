# 毛延寿画师传说详细资料

> 本笔记整理毛延寿传说的原始出处、文学演变及学界争议，作为游戏前置叙事的素材基础。

---

## 一、最早出处：《西京杂记》原文

**作者**：东晋·葛洪（依托西汉刘歆之名）
**性质**：笔记小说，非正史

### 原文

> 元帝后宫既多，不得常见。乃使画工图形，案图召幸之。诸宫人皆赂画工，多者十万，少者亦不减五万。**独王嫱不肯**，遂不得见。匈奴入朝，求美人为阏氏。于是上案图，以昭君行。及去，召见，**貌为后宫第一**，善应对，举止闲雅。帝悔之，而名籍已定。帝重信于外国，故不复更人。乃穷案其事，画工皆弃市，籍其家资皆巨万。
>
> 画工有**杜陵毛延寿**，为人形，丑好老少，必得其真；安陵陈敞，新丰刘白、龚宽，并工为牛马飞鸟众势，人形好丑，不逮延寿；下杜阳望亦善画，尤善布色。樊育亦善布色。**同日弃市**。

## 二、关键细节

| 条目 | 详情 |
|------|------|
| **涉及人数** | 共 **6名画工**同日被处死 |
| **毛延寿** | 杜陵人，**最擅长人物肖像**，"丑好老少，必得其真"，技艺最高 |
| **陈敞** | 安陵人，擅长牛马飞鸟 |
| **刘白、龚宽** | 新丰人，擅长牛马飞鸟 |
| **阳望** | 下杜人，擅长设色 |
| **樊育** | 擅长设色 |
| **受贿金额** | "多者十万，少者亦不减五万"——受贿规模巨大，抄家所得"皆巨万" |
| **昭君态度** | "**独**王嫱不肯"——只有她一人坚决拒绝行贿 |
| **元帝反应** | 见到真人"大惊"，但"**重信于外国**，故不复更人"——关键是**对外国不能失信** |

## 三、毛延寿的真实身份之谜

**正史中并无毛延寿其人。**

- 《汉书·元帝纪》《汉书·匈奴传》均未记载画工事
- 《后汉书·南匈奴列传》也未采用画工传说
- 毛延寿只存在于《西京杂记》及后世文学作品中

据此，学界普遍认为毛延寿是**文学形象而非历史人物**。

## 四、文学演变脉络

| 时期 | 作品 | 处理方式 |
|------|------|----------|
| 正史 | 《汉书》（班固） | 无画工事 |
| 笔记小说 | 《西京杂记》（葛洪） | 首次出现6画工同日弃市，毛延寿为首席 |
| 正史 | 《后汉书》（范晔） | **未采用**画工说，仅记"自愿请行" |
| 唐诗 | 杜甫《咏怀古迹》 | "画图省识春风面，环珮空归月夜魂"——同情昭君 |
| 唐诗 | 王安石《明妃曲》 | "意态由来画不成，当时枉杀毛延寿"——辩护 |
| 唐诗 | 白居易 | "自是君恩薄如纸，不须一向恨丹青"——归因于皇帝寡恩 |
| 唐变文 | 《王昭君变文》 | 匈强汉弱，毛延寿叛国献图 |
| 元杂剧 | 《汉宫秋》（马致远） | **毛延寿升为主角级反派**，中大夫，索贿不成→叛逃匈奴→献画像→煽动单于索人 |
| 明清 | 各类戏曲小说 | 毛延寿定型为奸臣形象 |

## 五、各代诗人对毛延寿的态度

### 杜甫《咏怀古迹·其三》
> 群山万壑赴荆门，生长明妃尚有村。
> 一去紫台连朔漠，独留青冢向黄昏。
> **画图省识春风面，环珮空归月夜魂。**
> 千载琵琶作胡语，分明怨恨曲中论。

**态度**：谴责画工误国，同情昭君命运。

### 王安石《明妃曲》
> 归来却怪丹青手，入眼平生几曾有。
> **意态由来画不成，当时枉杀毛延寿。**

**态度**：美人的神韵本就画不出来，杀毛延寿是冤枉的。格局更大——将批评矛头指向皇帝。

### 白居易《昭君怨》
> **自是君恩薄如纸，不须一向恨丹青。**

**态度**：归因于皇帝寡恩，画工只是替罪羊。

## 六、毛延寿形象在游戏中的使用建议

基于设计方案中的叙事定位（昭君自请出塞），毛延寿的处理建议：

1. **不正面出场**：序章和正文不需要出现毛延寿人物
2. **作为前置暗示**：可通过昭君在宫中的回忆、其他宫女的对话等侧面提及"画工丑化"的传说
3. **与王安石的态度共鸣**：游戏中可以暗示"意态由来画不成"——昭君的根本困境不在于画工，而在于深宫制度本身
4. **如做闪回**：若加入昭君在宫中的回忆闪回，可有一幕是"其他宫女贿赂画工，唯有她不语"的安静画面
