# 创作讨论记录

> 记录与老板（Wilfred）关于《昭君：一路向北》的讨论过程

---

## 2026年7月1日

### 项目启动
- **需求**：叙事类游戏的展示网页
- **方向**：交互式叙事体验页（玩家点击推进剧情、分支选择、结局展示）
- **题材**：昭君出塞

### 设计方案
老板提供了完整的《昭君：一路向北》设计方案V1.0，包含：
- 10章节结构（长安→渭水→秦岭→黄河→雁门关→草原→暴风雪→篝火→王庭→终章）
- 四种操作（前进、停留、对话、收藏）
- 记忆系统（《一路向北旅行册》）
- 声音系统为灵魂
- 技术栈：React + Vite + Phaser 3 + GSAP + Web Audio API

### 第一阶段决定
- **先整理故事内容/剧本，后开发**
- **结合史料讨论写法**

### 史料研究
搜索并整理了王昭君相关史料，关键发现：
1. 汉强匈弱，非屈辱和亲——呼韩邪是藩属自请联姻
2. 出塞路线更倾向雁门道（东路）
3. "和亲"一词在唐代才被泛化
4. 出塞后昭君曾上书求归，未被允许

### 语言风格
- **确定：现代散文风格**

### 序章方向
- 方向A（画轴自述体）→ 未采用
- 方向B（昭君独白）→ 未采用
- **方向C（大雁眼睛）→ 定稿** ✅
- 深秋，南飞大雁中一只停住，看向出城车队，马车中女子探出头回望长安。大雁调转方向跟随车队。
